# Balatro: Mewgenics Challenge Decks Mod

Crossover content mod for Balatro, featuring 8 new Decks based on the bosses of Mewgenics! These decks offer mostly detrimental effects in order to create unique challenges.

Compatible with the CardSleeves mod (https://github.com/larswijn/CardSleeves) to let true masochists combine or double-up challenge decks! Note, fun and sanity are not guaranteed when doing this.

Selecting a deck (or sleeve, but the decks have priority) will play the dynamic soundtrack of the area that the deck's boss is found in.

Also adds 4 detrimental enhancements unique to specific decks, and 2 additional final boss blinds.
